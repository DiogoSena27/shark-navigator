# Shark Navigator 🦈

An educational web game developed to train fundamental mouse mechanics for young coding students. Built from scratch with **HTML5** and **Phaser 3**.

## 🎯 Features
* **Precision Training:** Fast-paced target practice to improve reflex times and clicking accuracy.
* **Cargo Mode:** Drag-and-drop challenges with dynamic, moving drop zones.
* **Navigation:** Smooth cursor movement exercises, dodging falling obstacles without clicking.
* **Progressive Difficulty:** 5 levels per skill, scaling up to extreme difficulty, with a built-in timer for speedruns.
* **Local Storage:** Automatic progress saving in the browser.

## 🚀 Technologies Used
* HTML5 / CSS3
* Vanilla JavaScript (ES6)
* Phaser 3 Framework
